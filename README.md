# Face Recognition Server для Android 📸🚀

**Face Recognition Server** — это мощное Android-приложение для распознавания лиц в реальном времени с использованием передовых моделей машинного обучения (MediaPipe и FaceNet). Приложение может работать в трех режимах: локальное распознавание с камеры, дообучение модели на новых пользователях и встроенный HTTP-сервер для удаленного доступа по API.

## ✨ Ключевые возможности

*   **Многолицевое распознавание:** Одновременное отслеживание и распознавание до 5 лиц в кадре.
*   **Три режима работы:**
    1.  **Распознавание:** В реальном времени идентифицирует лица, отображая имена и уверенность.
    2.  **Дообучение:** Позволяет легко добавлять новых пользователей в базу, делая всего 12 снимков.
    3.  **HTTP-сервер:** Превращает устройство в сетевой API для распознавания лиц из других приложений.
*   **Визуальная обратная связь:**
    *   🟢 **Зеленая рамка** — лицо узнано (с именем и процентом).
    *   🔴 **Красная рамка** — лицо не найдено в базе.
    *   **Счетчик лиц** в углу экрана (например, "Лиц: 3 (узнано: 2)").
*   **Производительность:** Оптимизировано для работы на устройствах с ограниченными ресурсами (использование каскадов Хаара для детекции, кэширование результатов).
*   **REST API:** Полный набор эндпоинтов для интеграции с внешними системами.

## 🏗️ Архитектура проекта

Проект написан на **Kotlin** и следует лучшим практикам Android-разработки.

*   **Основные компоненты:**
    *   `MainActivity.kt`: Точка входа, управление камерой и UI.
    *   `FaceRecognitionEngine.kt`: Ядро приложения, объединяющее детекцию и распознавание.
    *   `FaceNetModel.kt`: Обертка для TensorFlow Lite модели FaceNet.
    *   `FaceRecognitionServer.kt`: Встроенный HTTP-сервер на базе Ktor.
    *   `FaceOverlayView.kt`: Кастомная View для отрисовки рамок и информации.

*   **Используемые технологии:**
    *   **CameraX** — для работы с камерой.
    *   **MediaPipe Tasks Vision** — для детекции лиц (модель `face_landmarker_v2`).
    *   **TensorFlow Lite** — для запуска модели **FaceNet**, извлекающей эмбеддинги лиц.
    *   **Ktor** — для встроенного HTTP-сервера.
    *   **Kotlin Coroutines** — для асинхронных операций.
