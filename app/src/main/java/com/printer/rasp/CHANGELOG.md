# История изменений Face Recognition Server

## [v1.2.0] - 2026-02-27
### 🚀 ПОДДЕРЖКА НЕСКОЛЬКИХ ЛИЦ И УЛУЧШЕНИЕ ПРОИЗВОДИТЕЛЬНОСТИ

#### 👥 МНОГОЛИЦЕВОЕ РАСПОЗНАВАНИЕ
1. **Одновременное отслеживание до 5 лиц**
    - Каждое лицо получает уникальный ID
    - Плавное движение с интерполяцией между кадрами
    - Автоматическое сопоставление лиц между кадрами
    - Счетчик лиц в углу экрана

2. **Визуальная индикация**
    - 🟢 Зеленые рамки для узнанных лиц
    - 🔴 Красные рамки для неизвестных
    - "?" для неопознанных лиц
    - Счетчик "Лиц: X (узнано: Y)"

3. **Трекинг лиц**
    - Алгоритм поиска ближайшего лица
    - Защита от потери отслеживания
    - Сброс трекинга при смене режима

#### ⚡ ОПТИМИЗАЦИЯ ПРОИЗВОДИТЕЛЬНОСТИ
1. **Ускорение обработки**
    - Уменьшен коэффициент расширения лица (2.5 → 1.8)
    - Оптимизирована конвертация изображений
    - Улучшена работа с памятью (своевременный recycle)
    - Кэширование результатов детекции

2. **Плавность анимации**
    - Коэффициент сглаживания 0.6
    - Защита от резких скачков рамки
    - Интерполяция координат между кадрами

#### 🎯 УЛУЧШЕНИЯ ПОЛЬЗОВАТЕЛЬСКОГО ОПЫТА
1. **Первый запуск**
    - Приветственное сообщение
    - Подсказка по добавлению пользователя
    - Информация о пустой базе желтым цветом
    - Скрыт список пользователей при пустой базе

2. **Интерфейс**
    - Уменьшен размер текста на рамках для нескольких лиц
    - Добавлен счетчик лиц в углу
    - Улучшена читаемость при множестве лиц
    - Цветовая индикация статуса базы

#### 🐛 ИСПРАВЛЕНИЕ ОШИБОК
1. **Стабильность**
    - Исправлена ошибка с smoothFactor
    - Улучшена обработка исключений
    - Добавлены проверки на null
    - Корректное освобождение ресурсов

2. **Точность распознавания**
    - Улучшено сопоставление лиц между кадрами
    - Оптимизирован поиск ближайшего лица
    - Добавлена защита от ложных срабатываний

#### 📊 ПРОИЗВОДИТЕЛЬНОСТЬ
| Режим | FPS | Лиц | Загрузка CPU |
|-------|-----|-----|--------------|
| Одно лицо | 8-12 | 1 | 40% |
| Несколько лиц | 6-8 | 3-5 | 60% |
| Дообучение | 10-12 | 1 | 45% |

#### 🔧 ТЕХНИЧЕСКИЕ УЛУЧШЕНИЯ
1. **Новые структуры данных**
    - `FaceWithEncoding` - лицо с эмбеддингом
    - `FaceTracker` - трекер для каждого лица
    - Оптимизированные коллекции

2. **Алгоритмы**
    - Поиск ближайшего лица по расстоянию
    - Плавная интерполяция координат
    - Сортировка лиц по размеру
    - Лимит на максимальное количество лиц

## [v1.0.0] - 2026-02-27
### Первый стабильный релиз
- Базовый функционал распознавания лиц
- Детекция через MediaPipe
- Извлечение признаков через FaceNet
- Дообучение на 12 образцах
- Встроенный HTTP-сервер
- REST API для удаленного доступа
- Темный интерфейс с анимациями
- Поддержка фронтальной/задней камеры

### Архитектура
- Kotlin + Android SDK
- CameraX для работы с камерой
- MediaPipe Tasks Vision
- TensorFlow Lite
- Ktor для HTTP-сервера
- Coroutines для асинхронности
- JSON для хранения данных
